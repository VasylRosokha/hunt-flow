# Feature: Dashboard

**Route:** `/dashboard`
**Access:** Protected

The main hub after login. Shows at-a-glance stats and recent activity.

## Stats Cards (top row)

- Total applications
- Response rate (% that moved past APPLIED)
- Applications this week
- Average days to response

## Charts

- **Status breakdown** — Donut/pie chart showing how many apps are in each status
- **Applications over time** — Bar chart showing applications per week for the last 8 weeks
- **Response funnel** — Funnel chart: Applied -> Screening -> Interview -> Offer

## Recent Activity

- Last 5 status changes across all applications with timestamps
- Example: "You applied to Frontend Developer at Productboard 2 days ago"

## Quick Actions

- "Add Application" button
- "Analyze Job Description" button
