# Feature: Application Detail

**Route:** `/applications/[id]`
**Access:** Protected

## Main Info Panel

- Company, position, location, salary, URL, platform
- Current status with color badge
- Status change buttons: quick-action buttons to move to next logical status

## AI Analysis Panel (if job description was provided)

- Match score (0-100) with visual meter
- Matching skills (green checkmarks)
- Missing skills (red gaps with suggestions)
- Key requirements extracted from job description
- "Generate Cover Letter" button

## Status History Timeline

- Visual timeline showing each status change with dates
- Example: "Saved -> Applied (March 15) -> Screening (March 20) -> Interview (March 25)"

## Notes Section

- Free-text area for user notes
- Contact person info

## Cover Letters

- List of generated cover letters for this application
- Copy to clipboard button on each
