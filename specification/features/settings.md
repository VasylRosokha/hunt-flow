# Feature: Settings

**Route:** `/settings`
**Access:** Protected

## Profile

- Name
- Email (read-only, from OAuth)
- Profile picture (from OAuth)

## Skills Management

- Add/remove skills from a tag-input component
- Pre-populated suggestions: React, TypeScript, JavaScript, Node.js, etc.
- These skills are used for AI matching

## Preferences

- Default platform for new applications
- Follow-up reminder threshold (default: 7 days)
